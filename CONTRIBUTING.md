# 贡献指南

欢迎通过 Issue 或 Pull Request 参与本项目。

## Issue 提交

- Bug 报告请附带复现步骤、运行环境、报错日志和最小输入样例。
- 功能建议请先说明业务场景、预期行为和对现有流程的影响。
- 涉及测距精度的问题，建议同时提供机型、分辨率、是否开启 RTK、样本图像和真值获取方式。

## Pull Request 要求

- 修改前请先阅读仓库现有 README 与 `docs/zh/` 下的说明文件。
- PR 描述应写明改动背景、核心变更点、验证方法和影响范围。
- 如变更影响用户使用方式、参数格式或接口行为，请同步更新 README 或相关文档。
- 请勿提交与当前任务无关的大范围格式化或重构。

## 数据与验证

- 示例数据应尽量脱敏，不要提交受限或敏感原始数据。
- 与精度相关的改动，建议附带独立验证结果或最小复现实验。
- 若新增机型支持，建议同时补充内参来源、标定方法和验证结论。
